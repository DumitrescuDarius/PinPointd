# AI Chat Server

A simple Express server ready for AI chat implementation.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

The server will run on port 3002 by default.

## API Endpoints

- `GET /` - Server status
- `GET /health` - Health check
- `POST /api/chat` - Chat endpoint (ready for AI implementation)

## Environment

- Node.js
- Express
- CORS enabled
